üìñ Description
---------------
This project is an AI-powered chatbot built using React and Next.js. The chatbot utilizes the Google GenAI API to generate human-like responses to user input. The application features a simple and intuitive interface, allowing users to engage in conversations with the chatbot. The chatbot's primary function is to understand and respond to user queries, providing valuable insights and information on a wide range of topics.

The chatbot's architecture is designed to be scalable and flexible, allowing for easy integration with various AI models and services. The application is built using a modular approach, with separate components for the chat interface, message rendering, and AI-powered response generation. This modular design enables developers to easily customize and extend the chatbot's functionality to meet specific use cases and requirements.

The project's goal is to demonstrate the potential of AI-powered chatbots in enhancing user engagement and experience. By leveraging the capabilities of AI and machine learning, the chatbot can provide personalized and context-aware responses, making it an ideal solution for various applications, such as customer support, language translation, and content generation. The project's codebase is well-structured and maintainable, making it easy for developers to contribute and build upon the existing functionality.

üìÅ Project Structure
---------------------
The project's structure is organized into the following folders and files:
* `components`: This folder contains the React components used in the application, including the `ChatInterFace` and `Message` components.
* `pages`: This folder contains the application's pages, including the `Home` page.
* `public`: This folder contains the application's static assets, such as images and fonts.
* `styles`: This folder contains the application's CSS styles, including the `globals.css` file.
* `utils`: This folder contains utility functions and helper modules used throughout the application.
* `route.js`: This file contains the API route for the chatbot's backend.
* `page.js`: This file contains the React component for the `Home` page.
* `globals.css`: This file contains the application's global CSS styles.
* `layout.js`: This file contains the React component for the application's layout.
* `package.json`: This file contains the application's dependencies and scripts.

‚öôÔ∏è How to Run
----------------
To run the application, follow these steps:
1. Clone the repository using `git clone`.
2. Install the dependencies using `npm install` or `yarn install`.
3. Create a `.env.local` file and add your Google GenAI API key.
4. Run the application using `npm run dev` or `yarn dev`.
5. Open a web browser and navigate to `http://localhost:3000` to access the application.

To build the application for production, run the following command:
```bash
npm run build
```
To start the application in production mode, run the following command:
```bash
npm run start
```
